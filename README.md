# ftp_client for linux and macos
### Usage
- $ make
- $ bin/client
### Command
- open ip_address (default control port is 21)
- cd pathname
- !cd pathname: client working directory.
- ls [-l]
- !ls: client working directory.
- syst: return server system type
- get pathname
- mget p1 p2 p3
- put pathname
- mput p1 p2 p3
- mkdir pathname
- quit
